# Telegram Cloud CLI 🚀

A Python-based Command Line Interface (CLI) tool that lets you use your Telegram account's **Saved Messages** as unlimited cloud storage. Powered by the **Telethon** library, it allows you to upload, download, update, and list files directly from your terminal.

---

## 📂 Project Structure

```text
TelegramCloud/
│
├── main.py                 # The core CLI application entrypoint
├── requirements.txt         # Python package dependencies
├── .env                    # Environment variables (API credentials)
├── .gitignore              # Files ignored by git (e.g., .venv, credentials)
│
# Autogenerated files after setup/run:
├── .venv/                  # Python virtual environment (recommended)
├── anon.session            # Telethon session database (keeps you authenticated)
└── anon.session-journal    # Telethon session journal database
```

---

## ✨ Features

- **⬆️ Upload Files:** Upload any local file with optional captions and a visual progress bar.
- **⬇️ Download Files:** Download files using their Telegram message IDs.
- **🔄 Update/Replace Files:** Replace files inside existing messages with newer versions without creating new messages.
- **📋 List Files:** Print a neat list of all files stored in your "Saved Messages" along with their IDs.
- **⏳ FloodWait Handling:** Automatic waiting and retrying when Telegram's API rate limits (`FloodWaitError`) are triggered.

## 🛠️ Setup & Installation

### 1. Prerequisites
- **Python 3.10** or higher installed.
- **Telegram API Credentials (`API_ID` & `API_HASH`)**:
  Follow these detailed steps to generate your credentials:
  1. Open your web browser and go to [my.telegram.org](https://my.telegram.org/).
  2. Enter your phone number in international format (e.g., `+1234567890`) and click **Next**.
  3. You will receive a confirmation code **inside the Telegram app** (sent by the official Telegram account, not SMS). Copy the code, paste it on the website, and click **Sign In**.
  4. Go to **API development tools**.
  5. In the **Create new application** form, fill in:
     - **App title**: Choose a name (e.g., `Telegram Cloud CLI`).
     - **Short name**: Choose a short identifier (e.g., `tgcloudcli`).
     - **Platform**: Select any option (e.g., `Other` or `Desktop`).
     - You can leave the URL and description empty.
  6. Click **Create application**.
  7. Copy the **App api_id** and **App api_hash** shown on the screen. These will be placed in your `.env` file.

### 2. Installation Steps

1. **Clone/Navigate** to the project directory:
   ```bash
   cd TelegramCloud
   ```

2. **Create and Activate a Virtual Environment** (recommended):
   - **Windows (PowerShell):**
     ```powershell
     python -m venv .venv
     .venv\Scripts\Activate.ps1
     ```
   - **Linux / macOS:**
     ```bash
     python3 -m venv .venv
     source .venv/bin/activate
     ```

3. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Configure Environment Variables:**
   Create a file named `.env` in the root directory and add your API keys:
   ```env
   API_ID = your_api_id_here
   API_HASH = your_api_hash_here
   ```

## 🚀 How to Use

1. **Run the Application:**
   ```bash
   python main.py
   ```

2. **Authenticate (First Time Only):**
   When running the tool for the first time, Telethon will ask you to enter:
   - Your **Phone Number** (with country code, e.g., `+84...`).
   - The **Verification Code** sent to your Telegram account.
   - Your **Two-Factor Authentication (2FA) password** (if enabled).

   This creates the local `anon.session` file, so you won't need to log in again on future runs.

3. **Interactive Menu Options:**
   Once logged in, you will be presented with a menu:
   - **`1. Upload a file`**: Provide the path to a local file (e.g., `world.zip`) and an optional caption.
   - **`2. Download a file`**: Retrieve a file by typing its corresponding message ID and target output path.
   - **`3. Update / replace a file`**: Replaces the file payload of a specific message ID with a new local file.
   - **`4. List all files`**: Displays a table of all files (ID and Filename) in your "Saved Messages".
   - **`5. Exit`**: Safely closes the Telethon client and exits.

## ⚠️ Disclaimer & Limitations

- **Telegram Terms of Service:** This project is intended for personal backup and educational use. Using Telegram as a CDN or abusing its storage might violate their [Terms of Service](https://telegram.org/tos) and could lead to rate limits or account restriction/suspension.
- **Upload File Limits:** 
  - Standard Users: Up to **2.0 GB** per file.
  - Telegram Premium Users: Up to **4.0 GB** per file.
- **Saved Messages Dependency:** Do not manually delete messages inside your "Saved Messages" chat if you still wish to fetch them via their message IDs.
